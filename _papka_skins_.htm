Папка Skins

Папка содержит скины, поставляемые с GenHelp. Пользователь может разработать собственный скин.

Файл

Назначение

skin.ini

Файл-описатель всех скинов, имеющихся в составе GenHelp.

 

Далее приводится типовая структура папки скина. В качестве примера приведен скин Aero_eng. Все файлы скина могут быть изменены или заменены пользователем (с соблюдением определенных правил, напр., копию файла следует поместить в папку Update, затем внести в него необходимые изменения). В имене скина присутствует язык, для которого скин предназначен (напр., Aero_eng ‑ для генерации ЭС на английском языке).
