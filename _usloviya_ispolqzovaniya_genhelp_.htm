Условия использования GenHelp

Программа LUXOFT Generic Help Workshop program (GenHelp) разрешена к использованию для технических писателей компании Luxoft или для лиц, прошедших обучение на курсе для технических писателей в учебном центре Luxoft.

Для работы с программой пользователь должен иметь следующие навыки, владение которыми характерно для технических писателей:

•   знать основы HTML и JavaScript;

•   уметь кодировать вызов утилит командной строки и команд ОС в bat-файлах;

•   уметь работать с популярными пакетами графики (например, программа Microsoft Visio удобна для разработки графических элементов управления Help, таких, как кнопка Contents).
