GenHelp

Настоящая электронная справка разработана для пользователя системы GenHelp.

Программа Luxoft Generic Help Workshop (далее сокращенно именуемая GenHelp) является пакетом программного обеспечения, предназначенного для автоматизированного преобразования исходного документа в формате Word (DOC) в набор файлов, представляющих собой комплекты электронной документации в форматах HTML, CHM и DOC (файл в формате Word, предназначенный для печати документа).

Целевой пользователь пакета - технический писатель, хорошо владеющий Word и имеющий навыки разработки HTML-страниц.

Для получения сведений о порядке работы со справкой щелкните на кнопке Помощь.
