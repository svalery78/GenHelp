Предметный указатель

Предметный указатель имеет следующую структуру:

•   Заголовок статьи (уровень 1)

•   Список индексных полей, связанных со статьей (уровень 2)

•   Список секций (страниц), содержащих индексное поле (уровень 3, гиперссылки)

Для того чтобы найти требуемую статью (приведен пример для браузера Microsoft Internet Explorer):

1.     Щелкните в любом месте фрейма, содержащего индекс для того, чтобы перевести на него фокус ввода

2.     Нажмите комбинацию клавиш Ctrl+Home для перехода в начало списка

3.     Найдите статью визуально, пользуясь при необходимости бегунком фрейма

4.     Для поиска статьи в фрейме:

•    Выполните команду в меню браузера Edit > Find… (или нажмите комбинацию клавиш Ctrl+F)

•    Наберите текст для поиска в поле ввода Find, Щелкните на кнопке Find Next

•    Выполняйте поиск необходимое число раз.

Для просмотра содержания электронной справки Щелкните на кнопке меню Содержание.
